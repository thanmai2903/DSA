# Selection Sort

**Algorithm:**

-   Repeatedly find the minimum element from the unsorted part and move it to the beginning.

**Time Complexity:**

-   Average and Worst-case: O(n^2)
-   Best-case: O(n^2)

**Space Complexity:**

-   O(1) (in-place sorting)

**C++ Code:**

```cpp
#include <iostream>
#include <vector>

void selectionSort(std::vector<int>& arr) {
    int n = arr.size();
    for (int i = 0; i < n - 1; ++i) {
        int minIndex = i;
        for (int j = i + 1; j < n; ++j) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }
        std::swap(arr[i], arr[minIndex]);
    }
}

int main() {
    std::vector<int> arr = {5, 1, 4, 2, 8};
    selectionSort(arr);

    for (int num : arr) {
        std::cout << num << " ";
    }
    std::cout << std::endl;

    return 0;
}
```
